# Lion Tech Mídia — Quickstart

## Instalar
npm install

## Rodar em desenvolvimento
npm run dev

## Build
npm run build

## Notas
- Tailwind já configurado.
- Firebase Firestore usado nos collections: tvPoints, clients, payments.
- Para produção (Vercel/Netlify), garanta que o build esteja apontando para SPA (Single Page App).
